    關於 Tailwind

    • Tailwind 官方文件 v.s. Bootstrap

      • Utility 類型(CSS 集合)框架 v.s. 組件類型(CSS + js 的)框架
        • 前者
          • 優: 彈性，可使用自訂的 css 命名
          • 缺: 需要額外引入 js 腳本
        • 後者
          • 優: 能便利快速的使用組件
          • 缺: 客製化不便

    • 從原先的 Boostrap 轉入 Tailwind 花了許多時間

      • 邏輯類似，但要求更多的 class
      • 更精細的設計意味著需要更多心力思考平常不會注意的細節
